```md
- 主题：<一句话>

- 最终产物：<页面、脚本、配置或站点>

## 起点与前提

- <初始状态>
- <账号、域名、仓库、平台等前提>

## 关键步骤

1. <步骤 1>
2. <步骤 2>
3. <步骤 3>

## 关键命令与路径

- 命令：`<command>`
- 路径：`<path>`

## 关键决策

- <为什么选当前方案>

## 踩坑与修复

- 问题：<症状>
  修复：<处理方式>

## 仍需研究或确认

- <可留空>
```
